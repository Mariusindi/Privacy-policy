    <h2>Norsk</h2>

    <p><strong>1. Ingen innsamling av personopplysninger</strong><br>
    Middagsrulett® (heretter referert til som DishSpin internasjonalt) samler ikke inn, lagrer ikke, og overfører ingen personopplysninger fra brukerne. Appen krever ingen innlogging eller opprettelse av brukerkonto.</p>

    <p><strong>2. Lokal lagring</strong><br>
    All data som legges inn i appen (oppskrifter, ingredienser, handlelister og ukeplaner) lagres utelukkende lokalt på brukerens egen enhet. Utvikleren har ingen tilgang til dine data.</p>

    <p><strong>3. Tredjepartstilgang og deling</strong><br>
    Appen inneholder ingen sporingsverktøy eller analyseverktøy fra tredjeparter. Ingen data deles med eksterne firmaer eller servere.</p>

    <p><strong>4. Google Play / Kjøp i app</strong><br>
    Transaksjoner håndteres direkte av Google Play. Appen mottar kun bekreftelse på kjøp for å låse opp funksjoner. Ingen betalingsinformasjon lagres i appen.</p>

    <p><strong>5. Kamera og bildebruk</strong><br>
    Appen kan be om tilgang til kamera for OCR (lese tekst fra bilder).<br><br>

    Appen lar også brukeren frivillig legge til bilder til retter. Disse bildene lagres kun lokalt på enheten og brukes kun i appen. De deles ikke med utvikler eller tredjeparter, og sendes ikke over internett.<br><br>

    Bilder kan bli inkludert i enhetens sikkerhetskopi (for eksempel Android backup), avhengig av enhetens innstillinger.</p>
